## Webtime

Here is an easy and quick way to make an Android “app” for any website. The application is just a wrapper around a WebView.

#### Instruction:
1) Get Android Studio Portable from https://portapps.io/app/android-studio-portable/ (if you dont have Android Studio already)

2) Download the project zip file and open in ASP

3) edit line 29 in MainActivity.java to set your own URL in mainURL

4) edit line 6 in build.gradle to set you own applicationID

5) edit line 2 in strings.xml to give your app a name 

6) put whatever image you want to use as a logo in app/res/drawable/ and name it as "icon.png" and "icon_round.png"
